NoReflectionUnusableEdgeSupplier 

class NoReflectionUnusableEdgeSupplier[T] extends Supplier[T] with Serializable

Any Edge type needs a supplier to avoid issues discussed in DefaultWeightedEdgeSupplier, but we never want this one to be called. Instead, instances of T should be constructed manually. This should be used in cases where T does not have a no-arg constructor.
