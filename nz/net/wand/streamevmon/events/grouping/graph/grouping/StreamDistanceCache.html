StreamDistanceCache 

class StreamDistanceCache extends CheckpointedFunction with Serializable

Keeps track of the distances between measurement streams. Uses DistanceBetweenStreams to calculate the distance. When an unrecognised stream is added, relevant distances are recalculated. No calculations occur if a recognised stream is added.
