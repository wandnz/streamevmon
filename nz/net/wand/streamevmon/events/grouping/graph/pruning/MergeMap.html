MergeMap 

class MergeMap[K, V] extends CheckpointedFunction

A Map that allows defining multiple keys per value, but is still accessed in the usual key-first manner. The underlying map contains only a single instance of each value.

If two keys are created with the same value, then any subsequent updates to either key will also update the value associated with the other key.
