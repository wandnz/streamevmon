  20.  def toMathString: String

    Creates a mathematical representation of the ParameterSpec.

    Creates a mathematical representation of the ParameterSpec. For Constants, we just return the value. For regular ParameterSpecs, we return the name.

    For a ModifiedSpec, we add the names of the SpecModifiers in order, surrounding each level of modification with brackets to preserve the order the operations are applied in.

    For example, a ModifiedSpec that first divides its value by two, then subtracts one should be represented as (x / 2) - 1.
