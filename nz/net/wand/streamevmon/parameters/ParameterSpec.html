ParameterSpec 

Companion object ParameterSpec

case class ParameterSpec[T](name: String, default: T, min: Option[T], max: Option[T])(implicit ordering: Perhaps[Ordering[T]]) extends Serializable with Product

Specifies the valid range of values for a parameter.

T

The type of the parameter's value. Note that many features, such as random generation of ParameterInstances, rely on T being an Int, Long, Double, or Float. Other types will need additional handling that is not yet implemented.

name

The full name of the parameter. For example, detector.changepoint.triggerCount, not just triggerCount.

default

The default value for the parameter.

min

The minimum value of the detector, or None for unordered types.

max

The maximum value of the detector, or None for unordered types.

  20.  def toMathString: String

    Creates a mathematical representation of the ParameterSpec.

    Creates a mathematical representation of the ParameterSpec. For Constants, we just return the value. For regular ParameterSpecs, we return the name.

    For a ModifiedSpec, we add the names of the SpecModifiers in order, surrounding each level of modification with brackets to preserve the order the operations are applied in.

    For example, a ModifiedSpec that first divides its value by two, then subtracts one should be represented as (x / 2) - 1.
