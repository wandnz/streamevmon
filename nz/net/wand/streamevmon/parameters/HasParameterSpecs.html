HasParameterSpecs 

Companion object HasParameterSpecs

trait HasParameterSpecs extends AnyRef

Allows a configurable class to expose its parameters in the form of a list of ParameterSpecs, as well as a list of associated constraints.

The parametersAreValid function can check if all the constraints of an implementing class are satisfied by a given set of parameters.
