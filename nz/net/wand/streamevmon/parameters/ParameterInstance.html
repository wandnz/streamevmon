    This package allows detectors to specify the parameters they accept, and any relational restrictions between them.

    This package allows detectors to specify the parameters they accept, and any relational restrictions between them. For example, you can specify that parameter A can be any integer between 0 and 100, and that parameter B must be less than half of parameter A's value.

ParameterInstance 

Companion object ParameterInstance

case class ParameterInstance[T](spec: ParameterSpec[T], value: T)(implicit ordering: Perhaps[Ordering[T]]) extends Serializable with Logging with Product

A particular value for a parameter specified by a ParameterSpec.

T

See ParameterSpec for details.

spec

The ParameterSpec for this instance.

value

The particular value of the parameter. Note that for ordered types, this should be between spec.min and spec.max. This is not enforced in this class, so it is possible to create an invalid ParameterInstance.
