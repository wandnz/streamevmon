PollingEsmondSourceFunction 

class PollingEsmondSourceFunction[EsmondConnectionT <: AbstractEsmondConnection, EsmondDiscoveryT <: AbstractEsmondStreamDiscovery] extends RichSourceFunction[RichEsmondMeasurement] with HasFlinkConfig with CheckpointedFunction with Logging

Outputs data from an Esmond API host.

Discovers the streams to query by passing the configuration to an instance of AbstractEsmondStreamDiscovery constructed using discoveryBuilder.

Configuration passed via HasFlinkConfig's overrideConfig gets passed to the StreamDiscovery and the EsmondConnection.
