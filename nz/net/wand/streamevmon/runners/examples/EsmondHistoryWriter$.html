    YamlDagRunner is the default entrypoint. It will run every supported detector according to the program config.

EsmondHistoryWriter 

object EsmondHistoryWriter extends Logging

Tools for writing historical Esmond measurements to a local archive.

This doesn't write anything to disk until all the measurements are queried, which can cause very high memory usage when archiving longer durations. It is recommended to provide a higher-than-default -Xmx argument to the JVM.

The data is written in four formats: Two JSON, two Java-Serialization. Each of these are written both as a large single blob, and as many files: one file for each stream, placed into sensible folders.
